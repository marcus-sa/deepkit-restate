## Prerequisites
A basic understanding of [TypeScript](https://www.typescriptlang.org), [Deepkit](https://deepkit.io) and [Restate](https://restate.dev)

Install the required dependencies
<Tabs>
  <Tab title="bun">
    ```sh
    $ bun add deepkit-restate
    ```
  </Tab>
  <Tab title="pnpm">
    ```sh
    $ pnpm add deepkit-restate
    ```
  </Tab>
  <Tab title="yarn">
    ```sh
    $ yarn add deepkit-restate
    ```
  </Tab>
  <Tab title="npm">
    ```sh
    $ npm install deepkit-restate
    ```
  </Tab>
</Tabs>
